# 医学文本结构化  - 测试教程
此工具包的结构化分类范围为：孕周、孕次、高危因素、胎位、结果、阿氏评分。

## 环境

- Python 3.6
- TensorFlow 1.14.0
- numpy
- scikit-learn
- scipy

## 运行过程
1，将`预处理`句子放在`data/lianyin.txt`中。  
2，跳转到`run_cnn.py`文件并执行。即`Run run_cnn`
